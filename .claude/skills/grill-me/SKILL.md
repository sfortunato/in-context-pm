description: Interview the user relentlessly about a plan or design until reaching shared understanding, resolving each branch of the decision tree. Use when user wants to stress-test a plan, get grilled on their design, or mentions "grill me".
---

Interview me relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one. For each question, provide your recommended answer.

If a question can be answered by exploring the codebase, explore the codebase instead.

Use AskUserQuestion popups to ask questions — batch up to 4 related questions per popup. This keeps the flow tight and avoids wall-of-text conversations. Only fall back to plain text questions when you need freeform narrative context that doesn't fit the popup format.
